Welcome to 4Clojure

4Clojure is a resource to help fledgling clojurians learn the language through interactive problems.
The first few problems are easy enough that even someone with no prior experience should find the learning curve forgiving.
See 'Help' for more information.

Let's Contribute!

Is this site written in Clojure? Absolutely! This site was created using a variety of open source Clojure (and Java) libraries. In fact, the code for this site is itself open source. Once you've mastered the language, feel free to contribute something back to the community.
